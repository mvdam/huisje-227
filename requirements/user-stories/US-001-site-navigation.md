# US-001: Site Navigation

**As a** visitor,
**I want to** navigate between all pages of the site via a menu,
**so that** I can find relevant information about the property.

## Source

- [index.html](../../html/index.html#L63-L75)

## Feature Area

[Navigation](../01-navigation.md)

## Acceptance Criteria

- The navigation menu is visible on all pages
- The menu contains links to: Home, Het huisje, Aankomst & Vertrek, Tarieven & Voorwaarden, Het Park, Contact, and Reserveren
- All menu labels are in Dutch
- Each link navigates to the correct page
