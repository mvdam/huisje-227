# US-002: Active Page Indicator

**As a** visitor,
**I want** the current page highlighted in the menu,
**so that** I know where I am on the site.

## Source

- [index.html](../../html/index.html#L68)

## Feature Area

[Navigation](../01-navigation.md)

## Acceptance Criteria

- The menu item for the current page has a visual `active` state
- Only one menu item is marked active at a time
- The active state is applied on every page
- Menu labels are in Dutch on all pages
