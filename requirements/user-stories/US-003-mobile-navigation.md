# US-003: Mobile Navigation

**As a** mobile visitor,
**I want** a hamburger menu toggle,
**so that** I can access navigation on small screens.

## Acceptance Criteria

- A hamburger icon is displayed on mobile/small screen viewports
- Tapping the icon toggles the navigation menu open/closed
- The menu opens with a fade animation (400ms)
- The menu can be closed by tapping the toggle again
- The mobile navigation displays all menu labels in Dutch
