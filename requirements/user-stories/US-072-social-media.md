# US-072: Social Media Links

**As a** visitor,
**I want to** access the property's social media profiles,
**so that** I can follow them for updates.

## Source

- [index.html](../../html/index.html#L1033-L1039)

## Feature Area

[Footer](../08-footer.md)

## Acceptance Criteria

- Social media icons are displayed in the footer
- Each icon links to the property's actual social media profile
- Links open in a new tab
- Only platforms actually used by the property are shown
- Any accompanying text or labels are in Dutch
