# US-080: Dutch Language & Real Content

**As a** visitor,
**I want** all pages to be in Dutch with real property information,
**so that** the site feels professional and trustworthy.

## Source

- All HTML files

## Feature Area

[Template Cleanup](../09-template-cleanup.md)

## Acceptance Criteria

- All page titles read "Capfun De Bongerd 227" (or page-specific variant)
- All navigation labels are in Dutch across all pages
- All form labels and buttons are in Dutch
- All content text is in Dutch
- No English template text remains visible to visitors
- SEO meta tags (description, keywords, author) are filled in
