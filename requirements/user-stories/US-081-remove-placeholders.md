# US-081: Remove Placeholder Content

**As a** property owner,
**I want** all placeholder content removed,
**so that** the site represents my actual holiday property.

## Source

- All HTML files

## Feature Area

[Template Cleanup](../09-template-cleanup.md)

## Acceptance Criteria

- No lorem ipsum text is visible on any page
- No English template text remains visible to visitors
- No template person names ("John Doe", "Jean Smith", "Will Peters", etc.) appear
- No template addresses ("98 West 21th Street, New York") are shown
- No template phone numbers or email addresses remain
- No template room types or USD pricing is displayed
- No template blog posts about travel appear
- All image alt text is descriptive and in Dutch (no "Free website template" or "Image placeholder")
- The "Leadership" section from the about page is removed
- All replacement content is written in Dutch
