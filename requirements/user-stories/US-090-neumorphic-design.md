# US-090: Neumorphic Design Language

**As a** visitor,
**I want to** see a modern neumorphic design across the website,
**so that** the site feels contemporary, visually appealing, and distinctive.

## Feature Area

[Visual Design](../10-visual-design.md)

## Acceptance Criteria

- All cards, buttons, and form elements use soft outer shadows creating a raised/embossed appearance
- Input fields and textareas use inset shadows to appear pressed/debossed
- All UI components have rounded corners (minimum 12px border-radius)
- Shadows use a consistent top-left light source direction
- Hover and active states transition between raised and pressed neumorphic styles (200–300ms)
- Pure black (`#000`) and pure white (`#fff`) are not used; off-black and off-white tones are used instead
- The neumorphic effect is visible but not exaggerated — the design remains clean and modern
