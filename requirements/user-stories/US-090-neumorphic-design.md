# US-090: Capfun Brand Card-Based Design

**As a** visitor,
**I want to** see a modern card-based design matching the Capfun de Bongerd brand,
**so that** the site feels professional, fresh, and consistent with the park's identity.

## Feature Area

[Visual Design](../10-visual-design.md)

## Acceptance Criteria

- All cards, buttons, and interactive elements use standard card shadows (`3px 3px 9px`) for subtle depth
- Input fields and textareas have a 1px solid border with subtle inset shadow
- All UI components have rounded corners (15px border-radius)
- Shadows use a single-direction light source (top-left to bottom-right)
- Hover states transition to deeper shadow (`7px 7px 14px`) with 200–300ms ease
- The design uses clean white/cream surfaces with Capfun green (#5c8200) as the primary accent
- The overall effect is clean and modern, not heavy or over-shadowed
