# US-097: Responsive Design

**As a** mobile visitor,
**I want to** use the website comfortably on any screen size,
**so that** I can browse, read content, and interact with forms without horizontal scrolling or misaligned layouts.

## Feature Area

[Accessibility](../11-accessibility.md)

## Acceptance Criteria

- The website is usable on viewports from 320px to 2560px wide
- No horizontal scrollbar appears at any supported viewport width
- Content reflows into a single column on narrow viewports (<576px) without loss of information
- All touch targets (buttons, links, form controls) are at least 44×44px on mobile
- Images and media scale proportionally without overflowing their containers
- Base font size is at least 16px; text is readable without zooming on mobile devices
- Navigation collapses into a mobile-friendly menu on small screens
